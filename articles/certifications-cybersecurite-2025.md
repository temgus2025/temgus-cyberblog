# Certifications en cybersécurité : lesquelles choisir en 2025 ?

Le marché des certifications en cybersécurité est vaste et parfois confus. Quelle certification choisir pour débuter ? Pour progresser ? Pour se spécialiser ? Voici un guide complet et honnête.

## Pourquoi se certifier ?

Les certifications servent à :
- **Prouver vos compétences** de façon objective et vérifiable
- **Structurer votre apprentissage** avec un programme défini
- **Être reconnu** par les recruteurs qui filtrent souvent par certification
- **Négocier un meilleur salaire** : +10 à +25% selon les études

Mais attention : une certification sans pratique réelle ne vaut pas grand chose. Les meilleures certifications combinent théorie et labs pratiques.

## Les certifications par niveau

### Niveau Débutant

#### CompTIA Security+ ⭐⭐⭐⭐⭐
**La référence pour débuter.** Reconnue mondialement, exigée par de nombreuses offres d'emploi aux US et de plus en plus en Europe.

- **Format** : QCM + questions à performance (labs simples)
- **Durée de préparation** : 2-4 mois
- **Prix** : ~380€
- **Valide** : 3 ans (renouvellement par CEUs)
- **Pour qui** : toute personne débutant en cybersécurité

#### eJPT (eLearnSecurity Junior Penetration Tester) ⭐⭐⭐⭐
Certification pratique et abordable, idéale pour débuter le pentest.

- **Format** : examen pratique (72h pour compromettre un réseau de labs)
- **Prix** : ~200€
- **Pour qui** : débutants en pentest

#### Google Cybersecurity Certificate ⭐⭐⭐
Nouveau certificat accessible via Coursera, bien structuré pour les débutants absolus.

- **Prix** : ~50€/mois (Coursera)
- **Pour qui** : reconversion professionnelle, débutants absolus

### Niveau Intermédiaire

#### CompTIA CySA+ ⭐⭐⭐⭐
Spécialisée Blue Team et analyse de sécurité. Suite logique de la Security+.

- **Prix** : ~380€
- **Pour qui** : analystes SOC, Blue Team

#### CEH (Certified Ethical Hacker) ⭐⭐⭐
Certification EC-Council très connue mais critiquée pour son aspect trop théorique.

- **Prix** : ~1 200€
- **Pour qui** : ceux qui ont besoin du badge "CEH" pour certains marchés (moyen-orient, grandes entreprises)

#### PNPT (Practical Network Penetration Tester) ⭐⭐⭐⭐⭐
Créée par TCM Security, entièrement pratique et très respectée dans la communauté.

- **Format** : pentest complet d'un réseau d'entreprise simulé (5 jours) + rapport
- **Prix** : ~400€
- **Pour qui** : pentesteurs en progression

#### CompTIA PenTest+ ⭐⭐⭐
Bonne certification intermédiaire en pentest, moins reconnue que l'OSCP mais plus accessible.

### Niveau Avancé

#### OSCP (Offensive Security Certified Professional) ⭐⭐⭐⭐⭐
**La référence absolue en pentest.** Extrêmement respectée, entièrement pratique.

- **Format** : 24h pour compromettre un maximum de machines, puis 24h pour rédiger le rapport
- **Prix** : ~1 500€ (inclut 90 jours de labs)
- **Prérequis** : solides bases Linux, réseau, scripting
- **Pour qui** : pentesteurs confirmés visant l'excellence

#### CISSP (Certified Information Systems Security Professional) ⭐⭐⭐⭐⭐
La certification de référence pour les managers et RSSI. Reconnue mondialement.

- **Format** : examen adaptatif (100-150 questions)
- **Prérequis** : 5 ans d'expérience en sécurité
- **Prix** : ~700€
- **Pour qui** : RSSI, managers sécurité

#### CISM (Certified Information Security Manager) ⭐⭐⭐⭐
Alternative au CISSP, plus orientée management.

- **Pour qui** : managers, futurs RSSI

#### AWS Security Specialty / AZ-500 / GCP Security ⭐⭐⭐⭐
Certifications cloud des grands providers. Très demandées avec la migration cloud massive.

- **Prix** : ~300€ chacune
- **Pour qui** : ingénieurs cloud, architectes sécurité

## Parcours recommandés

### Parcours Pentest
```
Security+ → eJPT → PNPT → OSCP
```

### Parcours Blue Team / SOC
```
Security+ → CySA+ → Blue Team Labs → GCIH (SANS)
```

### Parcours Management / RSSI
```
Security+ → CISM → CISSP
```

### Parcours Cloud Security
```
Security+ → AWS Cloud Practitioner → AWS Security Specialty
```

## Les pièges à éviter

**❌ Certifications trop chères sans valeur reconnue**
Méfiez-vous des certifications peu connues vendues très cher. Vérifiez leur reconnaissance sur LinkedIn job offers.

**❌ Bachoter sans comprendre**
Les "dumps" (banques de questions) permettent de réussir l'examen sans rien apprendre. Contre-productif à terme.

**❌ Collectionner sans spécialiser**
Mieux vaut 2-3 certifications solides dans un domaine que 10 certifications superficielles.

**❌ Ignorer les certifications pratiques**
Les certifications purement théoriques (QCM) ont moins de valeur que les certifications avec labs pratiques.

## Ressources pour se préparer

- **TryHackMe** : parcours de préparation Security+, OSCP
- **HackTheBox Academy** : excellents modules techniques
- **Professor Messer** : cours gratuits Security+ sur YouTube
- **TCM Security** : cours abordables pour PNPT et bases pentest
- **SANS** : formation premium, certifications GIAC très respectées

## Conclusion

La meilleure certification est celle qui correspond à votre niveau actuel et votre objectif de carrière. Commencez par la Security+ pour avoir une base solide, puis spécialisez-vous selon votre appétence pour l'offensif, le défensif ou le management. Et surtout : pratiquez en parallèle — une certification sans expérience pratique ne vaut pas grand chose sur le terrain.
